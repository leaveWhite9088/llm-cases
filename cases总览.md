# cases总览

每个 Case 是一个完整的"场景"，模型在同一个 workspace 内分多个阶段推进。
当前版本聚焦**前端能力**，每个 Case 都是综合的真实任务，能力点由阶段拆分观察，而不是按能力切分 Case。

## 能力分类

| 一级能力   | 二级能力                                         |
| ---------- | ------------------------------------------------ |
| 编码能力   | 前端、后端、架构、调试、测试、部署、运维         |
| Agent 能力 | 规划、工具调用、记忆（长短期）、自我修正、长任务 |
| 知识能力   | 世界知识、垂域知识                               |
| 推理能力   | 逻辑、因果、概率、数学                           |
| 内容能力   | 写作、创意、设计、审美、视频策划                 |
| 多模态能力 | 图片理解、空间理解、3D 场景构建、界面理解        |
| 信息能力   | 检索、归纳、事实核验、来源比较                   |
