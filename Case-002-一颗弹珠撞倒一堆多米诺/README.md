## 技术约束（决定实现路线，先看这节）

## 物理核心（这是"看起来对"的验收标准，不是让你自己实现的细节）

物理引擎会处理大部分物理模拟。本节告诉你**验收时要确认的现象**——物理引擎能自然产生这些时，演示就成功；如果某些现象没出现，调整物理引擎参数（摩擦、密度、弹性）即可。

1. **重力驱动**：钢珠在斜坡上受重力分量驱动向下加速，斜面摩擦决定加速度。
   - **验收**：钢珠滚动有重量感——起始慢、加速、中段最快，不是匀速滑动。

2. **动量传递**：钢珠撞第一张牌 → 牌的顶部击打下一张牌的侧面 → 能量逐级传递。
   - **验收**：首次撞击有明显的力学冲击感（不是轻轻碰一下）；连锁过程有自然的能量衰减（不是无限传递）。

3. **间隔影响节奏**：骨牌间距是关键变量。间距过小 → 能量被吞掉；间距过大 → 连锁中断。合适间距（牌高 1/2 到 2/3）让连锁既有速度又有节奏。
   - **验收**：连锁过程有自然的节奏起伏（紧密段快、稀疏段慢），不是匀速倒下。

## 场景设置

### 环境氛围（任选一个方向并保持一致）
- **极简抽象**：纯色背景 + 一束柔和顶光 + 地面柔和阴影，主体突出
- **桌面玩具感**：浅木纹桌面 + 自然光（窗户光）+ 轻微景深
- **工业展示感**：深色背景 + 强侧光 + 明显阴影 + 金属反射
- 其他合理方向也可，但必须整场视觉统一

### 斜坡
- 倾斜的平面（角度 20–35°），钢珠从顶端释放
- 材质：金属（反射强）、木质（哑光）、塑料（中性），任选
- 斜坡底端与地面之间平滑过渡，让钢珠能滚下来撞到第一张牌

### 骨牌
- 经典的长方体（宽厚比约 2:1:4），可以是白底黑点、纯色、彩色、木纹等
- 至少 30 张组成完整连锁
- 骨牌要有体积感（有厚度、有重量感，不是纸片）
- 排布方式至少支持三种：直线、曲线（S 形或弧形）、分叉（Y 形）

### 钢珠
- 一个小球（直径约为骨牌宽度的 1/3 到 1/2）
- 材质：金属（高反射）、玻璃（透明）、塑料（中等光泽），任选
- 滚动时自身旋转（不只是平移）

### 地面
- 骨牌站立的平面
- 材质：木质、瓷砖、桌面、石板，任选，要与整体氛围一致
- 必须能接收阴影

## 动画流程

### 阶段 1：释放与下落（2–3 秒）
用户点击「开始」后，钢珠出现在斜坡顶端，从静止释放，在重力作用下沿斜坡向下加速滚动。

**视觉表现：**
- 滚动有速度感（起始慢、加速、中段最快）
- 钢珠自身旋转
- 镜头：跟随钢珠，或先远景后切近景

### 阶段 2：首次撞击（视觉高潮 #1，0.3–0.5 秒）
钢珠撞上第一张牌，第一张牌倒下击中第二张。

**视觉表现：**
- 撞击瞬间：可以有粒子效果（灰尘、火花任选其一，或都不要）
- 第一张牌的倒下要有力（不是轻飘飘的倒下）
- 镜头可以给一个轻微的"撞击反应"（微抖或微缩放）

### 阶段 3：连锁反应（核心高潮，5–15 秒）
骨牌逐张倒下，能量沿链条传递。

**视觉表现：**
- 节奏感：紧密段（间隔小、倒下快速）+ 稀疏段（间隔大、倒下有停顿），整体不要匀速
- 镜头跟随"倒下波前"（最新的那张正在倒的牌）
- 每张牌倒下时有音效（节奏感比音调变化更重要）
- 倒下过程中牌与地面碰撞产生灰尘或反光变化

**关键质量点：**
- 连锁不能中途断（间距要合理）
- 连锁不能"全段匀速"（要有节奏变化）
- 末端的牌倒下要有"最后一击"的感觉

### 阶段 4：尾声（视觉高潮 #2，1–3 秒）
所有骨牌倒下，场景归于平静。

**视觉表现：**
- 最后几张牌的倒下可以给一点时间停留
- 镜头可以从连锁全景切到一片倒下的骨牌特写
- 地面上的骨牌保持倒下状态，形成最终的视觉构图
- 重置按钮让场景回到初始状态

## 视觉与渲染要求

### 骨牌渲染
- 有材质感（不是纯色片），允许轻微反光、纹理、阴影
- 骨牌之间的间距视觉清晰
- 倒下后能看到骨牌的厚度和侧面

### 地面与阴影
- 地面接收骨牌和钢珠的阴影（柔和阴影也可）
- 阴影随物体运动实时更新

### 光照
- 至少一盏主光（方向性光，制造主阴影）+ 环境光（让暗部不至于全黑）
- 整体亮度让所有细节清晰可见（不要过暗）

### 镜头
- 跟随"倒下波前"是核心
- 不要求电影级运镜，但要有"镜头在跟踪事情发生"的感觉

## 交互控制

- **「开始」按钮**：触发钢珠释放动画
- **「重置」按钮**：恢复场景到初始状态（钢珠回到顶端、所有骨牌立起）
- **时间缩放滑块**：0.1x – 1.0x 慢动作控制，方便观察连锁细节
- **排布切换**：直线 / 曲线 / 分叉 三种排布，切换后自动重置
- **起点调整**：用户能拖动调整钢珠释放位置（在斜坡上的位置或高度）
- **视角切换**：至少 3 个预设视角（全景、侧面、特写），鼠标支持自由旋转缩放（OrbitControls 或等价方案）
