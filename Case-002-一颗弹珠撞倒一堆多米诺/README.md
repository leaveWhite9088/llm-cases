# Case-002 一颗弹珠撞倒一堆多米诺

> 一个完整物理动画场景，从单点效果到完整可玩版本，一次交付

## 场景

一颗钢珠从斜坡顶端释放，在重力作用下滚下斜坡，最终撞倒底部排布的多米诺骨牌，触发完整连锁反应。镜头跟随连锁过程，最后用户能调整起点、看慢动作、听音效。整个 Case 是一次性交付——模型需要在一个工作会话里把整个可玩的场景做完。

## 需求

### 必含元素

1. **物理引擎**：重力、摩擦、碰撞真实可信
2. **视觉**：构图清晰、骨牌有体积感、有背景层次
3. **连锁**：至少 30 张骨牌的完整连锁
4. **镜头**：跟随连锁过程，能看到完整倒下的张力
5. **节奏**：骨牌间隔有变化，能感受到节奏
6. **交互**：用户能调整弹珠起点位置、能在直线 / 曲线 / 分叉之间切换骨牌排布
7. **音效**：骨牌倒下有声音
8. **慢放**：用户能切换慢动作回放

### 技术约束

- 纯前端：HTML + CSS + JavaScript（不引入框架）
- 物理引擎推荐 matter.js；自实现也可，但必须物理可信
- Canvas 或 SVG 渲染任选
- 在 Chrome / Edge 最新版流畅运行（≥ 30fps）

### 不需要做的

- 真实 3D（2D 即可）
- 复杂的关卡编辑器（一次完整排布即可）
- 移动端适配（桌面浏览器跑通即可）
