## 物理核心

**必须使用一个成熟的 2D 物理引擎。** 这是本任务最关键的技术决策——本演示的核心挑战是"看起来对"的物理表现，一个稳定、参数可调的物理引擎能解决 90% 的工作；自实现碰撞检测、速度积分、约束求解（手写重力 + AABB）会把任务难度拉满且几乎必然失败。

- 推荐：**matter.js**（轻量、稳定、文档丰富、社区活跃，本任务首选）
- 也可以：planck.js（Box2D 的 JS 移植）、p2.js 等其他等价 2D 物理引擎

本演示的视觉效果依赖三个关键物理现象：

1. **重力驱动**：钢珠在斜坡上受重力分量驱动向下加速，斜面摩擦决定加速度大小。
2. **动量传递**：钢珠撞第一张牌 → 牌的顶部击打下一张牌 → 能量逐级传递。
3. **间隔影响节奏**：骨牌间距是关键变量。间距过小，能量被吞掉；间距过大，连锁中断。合适间距（牌高 1/2 到 2/3）让连锁既有速度又有节奏。

## 场景设置

### 环境氛围（任选一个方向并保持一致）
- **极简抽象**：纯色背景 + 一束柔和顶光 + 地面柔和阴影，主体突出
- **桌面玩具感**：浅木纹桌面 + 自然光（窗户光）+ 轻微景深
- **工业展示感**：深色背景 + 强侧光 + 明显阴影 + 金属反射
- 其他合理方向也可，但必须整场视觉统一

### 斜坡
倾斜的平面，钢珠从顶端释放。材质：金属感、木质哑光、塑料中性，任选。斜坡底端与地面之间平滑过渡，让钢珠能滚下来撞到第一张牌。

### 骨牌
经典的长方体（宽厚比约 2:1:4），可以是白底黑点、纯色、彩色、木纹等。**至少 30 张组成完整连锁**，排成一条直线。骨牌要有体积感（有厚度、有重量感，不是纸片）。

### 钢珠
一个小球（直径约为骨牌宽度的 1/3 到 1/2）。材质：金属感、玻璃质、塑料中性，任选。滚动时自身旋转（不只是平移）。

### 地面
骨牌站立的平面。材质：木质、瓷砖、桌面、石板，任选，要与整体氛围一致。必须能接收阴影。

## 动画流程

### 阶段 1：释放与下落（2–3 秒）
用户点击「开始」后，钢珠出现在斜坡顶端，从静止释放，在重力作用下沿斜坡向下加速滚动。

钢珠的滚动要有重量感——起始慢、加速、中段最快，不是匀速滑动。斜面摩擦决定加速度大小，斜面越粗糙滚得越慢。钢珠自身在旋转（不只是平移）。画面可以跟随钢珠，或先远景后切近景，给观众"追踪滚下来"的代入感。

### 阶段 2：首次撞击（视觉核心之一，0.3–0.5 秒）
钢珠以一定速度撞上第一张牌，第一张牌倒下击中第二张。

**关键细节：** 首次撞击必须有力学冲击感（不是轻轻碰一下）。撞击瞬间可以加粒子效果（灰尘、火花任选其一，或都不要）来强化冲击；画面可以给一个轻微的"撞击反应"（微抖）让观众感受到力度。第一张牌的倒下也要有力——不是轻飘飘的倒下，而是带着整个链条即将开始崩的预感。

### 阶段 3：连锁反应（视觉核心之二，5–15 秒）
骨牌逐张倒下，能量沿链条传递。这是整个演示最壮观的阶段。

**关键细节：** 节奏感是核心。骨牌间隔有变化——紧密段（间隔小、倒下快速）+ 稀疏段（间隔大、倒下有停顿），整体不要匀速。画面要跟随"倒下波前"（最新的那张正在倒的牌），让观众的眼睛跟着连锁走。每张牌倒下时有音效（节奏感比音调变化更重要）。倒下过程中牌与地面碰撞产生灰尘或反光变化，强化能量释放感。

连锁不能中途断（间距要合理）；连锁不能"全段匀速"（要有节奏变化）；末端的牌倒下要有"最后一击"的感觉。

### 阶段 4：尾声（1–3 秒）
所有骨牌倒下，场景归于平静。

最后几张牌的倒下可以给一点时间停留，让观众感受到"最后一击"。画面可以从连锁全景切到一片倒下的骨牌特写——所有骨牌横躺，形成最终的视觉构图。重置按钮让场景回到初始状态。

## 视觉与渲染要求

### 骨牌渲染
有材质感（不是纯色片），允许轻微反光、纹理。骨牌之间的间距视觉清晰。倒下后能看出骨牌的厚度感。

### 阴影
骨牌和钢珠在地面投出阴影。阴影随物体运动实时更新。

### 明暗
通过明暗对比营造空间感（不需要真实光照计算）。整体亮度让所有细节清晰可见。

### 画面跟随
跟随"倒下波前"是核心。不要求电影级运镜，但要有"画面在跟踪事情发生"的感觉。

## 性能注意

- Chrome / Edge 最新版流畅运行（目标 60fps，30fps 是底线）
- 至少 30 张骨牌的连锁必须流畅
- 材质应共享（不要每张牌一个独立材质实例）
- 不要在动画过程中频繁创建和销毁动态对象

## 交互控制

- **「开始」按钮**：触发钢珠释放动画
- **「重置」按钮**：恢复场景到初始状态（钢珠回到顶端、所有骨牌立起）
- **时间缩放滑块**：0.1x – 1.0x 慢动作控制，方便观察连锁细节
- **起点调整**：用户能拖动调整钢珠释放位置（在斜坡上的位置或高度）
